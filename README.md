# Smartforce Settings

This repository contains several core elements used in the Smartforce Applications. This includes:

- SFSettings Component
- The Tours module
- Core Components
- Helpers
- Hooks
- Constants
- Contexts
- Models
- Services

## Instalation

`npm install`

## Development

`npm start`

## SFSettings

To use the SFSettings component in an application, it needs some contexts to be added first in the component tree:

- ThemeTypeProvider
- SFThemeProvider
- MediaProvider
- TimezonesProvider
- UserProvider
- CustomerProvider
- SubscriptionProvider
- AreasProvider
